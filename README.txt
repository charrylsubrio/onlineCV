# CSS101 Exercise: Online CV Web Page

This repository contains the HTML and CSS source code for an Online CV Web Page, mimicking a resume but presented as a web page. The goal of this exercise is to gain hands-on experience with HTML and CSS to create a visually appealing and informative personal profile.

## Project Overview

The profile web page includes various sections to showcase personal information, educational background, work experience, skills, and other relevant details.
